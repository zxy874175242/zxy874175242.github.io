vue中height：100%失效

https://www.jb51.net/article/144589.htm

height：100%时，div的高度等同于父元素的高度。

``Helloworld.vue中最外层<div>的父元素为App.vue中的id为app的div，其父元素为index.html`中的、。要将所有的父元素的高度设置为100%。

2020.03.27

url属性：

  • 如果该css文件和北京图片在同一文件夹（或该文件夹的子文件夹），则直接写文件名（子文件夹名/文件名）。

  • 如果不在同一文件夹，则用../

    “../“表示上一级目录

    “./“ 表示同级目录

    “/“ 表示根目录开始

CSS background-repeat 属性：

  • repeat 默认。背景图像将在垂直方向和水平方向重复。
    repeat-x 背景图像将在水平方向重复。
    repeat-y 背景图像将在垂直方向重复。
    no-repeat 背景图像将仅显示一次。
    inherit 规定应该从父元素继承 background-repeat 属性的设置。
