url属性：

  • 如果该css文件和北京图片在同一文件夹（或该文件夹的子文件夹），则直接写文件名（子文件夹名/文件名）。

  • 如果不在同一文件夹，则用../

    “../“表示上一级目录

    “./“ 表示同级目录

    “/“ 表示根目录开始

CSS background-repeat 属性：

  • repeat 默认。背景图像将在垂直方向和水平方向重复。
    repeat-x 背景图像将在水平方向重复。
    repeat-y 背景图像将在垂直方向重复。
    no-repeat 背景图像将仅显示一次。
    inherit 规定应该从父元素继承 background-repeat 属性的设置。

Hexo标签与MD文件冲突问题

https://www.jianshu.com/p/6032a1a2dc25

在使用hexo g命令时，出现报错：

出现上述情况的原因是 Markdown 文件中的 `{{}}` 标签与 nunjucks 模板引擎的标签发生了冲突。 `{{}}`，`{# #}`，`{% %}` 等这些标签都是模板引擎的占位标签，如果 MarkDown 文件中包含这些标签，且不被 代码块包裹（单代码块不行）那么解析时就会把 MD 文件中的标签动态解析了，于是导致 MD 文件解析时报错。
